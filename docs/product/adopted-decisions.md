# 採用済み判断

この文書は、Issue ベースで採用済みになった重要判断を短くまとめた一覧である。  
長い議論の本文は各 Issue に残しつつ、採用事項だけをここで追えるようにする。

## Issue #1

- 問い・記録・振り返りのループを回し続けられることを重視する
- 家族内の軽量ユーザー切り替えを持つ MVP とする

## Issue #2

- AI は問いの作者ではなく、本人の問いの芽を整える役に寄せる
- `next_curiosity_text` のような任意入力を持つ
- 問い選択後に「記録項目を整える」ステップを持つ
- 記録項目は役割つき候補 + 本人選択にする
- 本人が手動で記録項目を追加できる
- 記録項目は観測値を優先し、集計値を直接持ち込まない
- 記録項目には「何を見るためか」と「あとでどう使うか」を持たせる

## Issue #3

- Issue は意思決定ログとして残す
- 現行仕様の正本は文書に再統合する
- 壊してはいけないことは短い別文書で明文化する
- AI に渡す最小コンテキストと文書更新ルールを定義する
