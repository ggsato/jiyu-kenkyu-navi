# 自由研究とは何か

この文書は、自由研究ナビの背後にある「自由研究の捉え方」をまとめた思想文書である。

## 1. 自由研究の中心

自由研究は、与えられた問いに正しく答えることではない。  
自分の願いから出発し、自分で問いを立て、小さく試し、記録し、振り返りながら、自分の地図を育てていく営みである。

したがって、自由研究の中心は次の循環にある。

1. 願いを持つ
2. 小さな問いを立てる
3. 試す・観る・集める
4. 記録する
5. 振り返る
6. 次の問いや次の一歩を決める

## 2. 出発点は願いである

自由研究は、最初から立派なテーマや完成された研究計画で始まるとは限らない。  
出発点にあるのは、多くの場合もっと素朴な願いである。

- もっと速くなりたい
- うまく育てたい
- なぜこうなるのか知りたい
- 失敗しない方法を見つけたい

テーマは、その願いをたどるなかであとから育っていく。

## 3. 小さな問いが必要である

願いはそのままでは広すぎることが多い。  
そこで、一回で試せる大きさまで切り分けた「小さな問い」が必要になる。

よい小さな問いは、次の条件を満たす。

- 一回で試せる
- 観察や記録ができる
- 次の行動につながる
- 願いとつながっている

## 4. 記録は前進の材料である

記録は単なる保存ではない。  
記録は、比較や関連の発見、問いの立て直し、次の一歩の判断材料になる。

自由研究では、問いが動かない時期や、記録しかできない時期がある。  
しかし、その時期も止まっているのではなく、材料をためている時期である。

## 5. 振り返りは問いを育てる

振り返りでは、少なくとも次を整理する。

- 何が分かったか
- 何が分からなかったか
- 次に何をしたいか

振り返りの役割は、答えを確定することではなく、現在地を見えるようにし、次の一歩を決めやすくすることである。

## 6. 軌跡も成果である

自由研究では、最終的な答えだけが成果ではない。  
どんな問いを立て、どんな記録を重ね、どう方向を変えてきたかという軌跡そのものが重要である。

自由研究ナビも、この軌跡を価値ある成果として扱う。
